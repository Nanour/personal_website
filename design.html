My Work - Game



Play Gomoku with Sound (Speech API, VB, photoshop)
Developed a Gomoku game with the Microsoft speech API and Visual Basic. Players can easily say ‘play’ to start the game, and speak two numbers(row and column) to locate the chessman. Since it is a traditional game, the interface has been designed in a very user-friendly way. Two characters on the left can easily show the players whose turn right now, and identify the color of the chessman. After one play win the game, a message with the results will pop-up on the screen, and say ‘play’ again to start another game.

Play Tetris with Your Body (Kinect API, C#, photoshop)
Developed a windows Kinect game Tetris which users can play it with their body. For example, both hands up to rotate the object, left hand up to move the object to left, two hands down to accelerate the speed of going down.

Memory Game(html,css,javascript)
A single memory game developed by javascript. The game will first ask users to input their name, after entering the name, the memory game starts. Click the square to flip to see whether two pictures match. Game over when all pictures are matched. Users playing history will be stored in cookie for users to review.

OpenGL single game(C, OpenGL)
Developed a 2D maze game, which users can drag the arrow in the maze to reach the destination. I have also created 3D environment for a 3D ball game and implemented bounding volume to detect collisions.
